MAP

MAP (Members Attending Parliament) tracks the attendance of Indian MPs in Lok Sabha (lower house).

Currently data for 15th Lok Sabha is presented. Data for 12 sessions has been included.

Class information

  • 1st class - 75% and above attendance
  • 2nd class - 65% to 75% attendance
  • 3rd class - below 65% attendance

Idea

Conceptualized this idea, thanks to a Tweet.

Data

Data has been accessed from Lok Sabha website. Data is available in CSV format. Data can potentially be corrupted at few places. Please report to @bkamapantula.

Data format

Id, MP Name, Number of days signed, Number of days missed, Gender, State, Party, Constituency

Code

Code for parsing the data out of the Lok Sabha website will be made available soon. To contribute, head to Github.
